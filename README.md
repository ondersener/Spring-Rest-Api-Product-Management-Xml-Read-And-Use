# Spring-Rest-Api-Product-Management-Xml-Read-And-Use
Spring Rest Api Product Management, Xml Read And Use - JPA

# Software Versions :
- Java 8
- Spring 2.7.14
- MySQL 8

# Application Services : 
- Admin Login
- Product Add
- Product List
- Product Delete
- Product Update
- TCMB Exchange Rates Read -Jsoup
- News List - Jsoup
